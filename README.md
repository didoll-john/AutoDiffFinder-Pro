# AutoDiffFinder-Pro
辅助进行《找不同》游戏，上传游戏界面截图，要求：

- 包含完整的对比区域
- 对比区域是上下结构

界面使用gradio，拖动截图到图片区域，提交即可得到对比结果。

优点是对截图几乎无要求，可以直接把整个游戏界面截图上传

缺点是速度慢

如果嫌慢可以去试试[AutoDiffFinder-Lite](https://github.com/didoll-john/AutoDiffFinder-Lite)
